# WEBSITE BÁN QUẦN ÁO ONLINE

## Giới thiệu
Đây là website bán quần áo online được xây dựng nhằm hỗ trợ khách hàng xem sản phẩm, đặt hàng và thanh toán trực tuyến.  
Hệ thống hỗ trợ quản lý sản phẩm, quản lý đơn hàng và quản lý khách hàng dành cho quản trị viên.
